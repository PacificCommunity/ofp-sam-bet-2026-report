# Managing report figures

1. Open the writing task in Kflow and launch RStudio.
2. View the available images in `bet-2026-report/Figures/`.
3. Upload or replace an image in that directory.
4. Edit `sections/Figures.qmd` only when its filename, caption, order, or
   inclusion changes.
5. Run `bash run.sh` from the repository root to render the PDF.
6. Commit and push the changed source and assets together.

The report is self-contained. It does not read a Results-job archive.
